# fak

Generate faker data on the terminal or in Bash scripts.

Installation:

```bash
$ cargo install fak
```

Usage:

```bash
$ fak "Hi %name! You just earned %u32 coins"
Hi Frederick Stanton! You just earned 242323242 coins
```
